### とりあえず動かしたい人向け使い方
* eclipseでhttps://github.com/bis5/resumegen.gitをプロジェクトとしてインポート
* Run -> Java Applicationでnet.bis5.resumegen.Bootstrapをメインクラスとして実行
* Webブラウザでhttp://localhost:8080/resumegenにアクセス
* 必要事項を入力して「作るよ！」ボタンをクリック
